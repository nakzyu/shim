## Introduction

Shim is a single-page application made in MERN stack.

## Features

- edit user profile & message

- edit & delete post content 

- like & unlike other's posts

## Technologies

### Client

- React
- Firebase

### Server

- Express
- MongoDB
- JWT
- Cloudinary
- Heroku
